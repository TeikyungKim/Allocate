# 동적 자산 배분 전략 — 비교 기준 상세 가이드

> 동적 전략은 "지금 어떤 자산이 좋은가?"를 **수치로 판단**하여 자동으로 공격/방어를 전환합니다.
> 이 문서는 각 전략이 **무엇을 기준으로, 어떻게 비교하는지** 상세히 설명합니다.

---

## 1. 비교의 핵심 도구: 모멘텀(Momentum)

동적 전략의 거의 모든 비교는 **모멘텀** — 즉 "최근 N개월간 얼마나 올랐는가?"에 기반합니다.

### 1-1. 단순 수익률 (Simple Return)

가장 기본적인 비교 도구입니다.

```
수익률 = (현재 가격 - N개월 전 가격) / N개월 전 가격
```

| 룩백 기간 | 의미 |
|-----------|------|
| 1개월 | 단기 추세 (노이즈 많음, 민감도 높음) |
| 3개월 | 단기~중기 추세 |
| 6개월 | 중기 추세 |
| 12개월 | 장기 추세 (안정적, 반응 느림) |

**예시:** SPY의 12개월 수익률이 +15%이고, EFA가 +8%이면 → SPY가 더 강한 모멘텀을 가진다고 판단합니다.

---

### 1-2. 가중 모멘텀 스코어 (13612W)

VAA, DAA, BAA 등 Wouter Keller 계열 전략이 사용하는 핵심 공식입니다.

```
13612W = 12 × (1개월 수익률) + 4 × (3개월 수익률) + 2 × (6개월 수익률) + 1 × (12개월 수익률)
```

| 가중치 | 기간 | 비중 |
|--------|------|------|
| 12 | 1개월 | 63.2% |
| 4 | 3개월 | 21.1% |
| 2 | 6개월 | 10.5% |
| 1 | 12개월 | 5.3% |

**핵심:** 최근 1개월에 63%의 비중을 두어, **최근 추세 변화에 빠르게 반응**합니다. 장기 수익률이 아무리 좋아도 최근 1개월이 급락하면 스코어가 크게 떨어집니다.

**예시 계산:**
```
SPY: 1M=+2%, 3M=+5%, 6M=+8%, 12M=+15%
13612W = 12×0.02 + 4×0.05 + 2×0.08 + 1×0.15
       = 0.24 + 0.20 + 0.16 + 0.15
       = 0.75

AGG: 1M=-1%, 3M=+1%, 6M=+2%, 12M=+3%
13612W = 12×(-0.01) + 4×0.01 + 2×0.02 + 1×0.03
       = -0.12 + 0.04 + 0.04 + 0.03
       = -0.01

→ SPY(0.75)가 AGG(-0.01)보다 압도적으로 높으므로, SPY를 선택
→ AGG는 음수이므로 "방어 전환" 트리거 가능
```

---

### 1-3. 단순 이동평균 (SMA)

GTAA 전략이 사용하는 추세 필터입니다.

```
SMA(10) = 최근 10개월 종가의 평균
```

**비교 방식:** "현재 가격 > SMA(10)이면 상승 추세, 아니면 하락 추세"

이 방식은 자산 간 상대 비교가 아니라, **각 자산이 독립적으로 자기 자신의 추세와 비교**하는 것이 특징입니다.

---

### 1-4. 평균 모멘텀 (Average Momentum)

가속 듀얼 모멘텀(ADM), 복합 절대 모멘텀(CAM) 등이 사용합니다.

```
평균 모멘텀 = (1개월 수익률 + 3개월 수익률 + 6개월 수익률) / 3
```

13612W처럼 가중치를 두지 않고, 여러 기간의 **단순 평균**을 사용합니다. 13612W보다 단기 변동에 덜 민감합니다.

## 2. 비교 방식의 3가지 유형

### 유형 A: 상대 모멘텀 (Relative Momentum)

> "여러 자산 중 가장 잘 나가는 놈을 고른다"

자산들의 모멘텀 스코어를 비교하여 **순위(Rank)**를 매기고, 상위 N개를 선택합니다.

```
SPY: 모멘텀 0.75  → 1위 ✅ 선택
EFA: 모멘텀 0.30  → 2위
EEM: 모멘텀 0.10  → 3위
AGG: 모멘텀 -0.01 → 4위

Top-1 전략이면 → SPY 100%
Top-3 전략이면 → SPY + EFA + EEM 각 33.3%
```

**사용 전략:** Dual Momentum, VAA, DAA, BAA, PAA 등 대부분

---

### 유형 B: 절대 모멘텀 (Absolute Momentum)

> "이 자산이 오르고 있는가, 아닌가?" (Yes/No)

모멘텀 스코어가 **양수인지 음수인지**만 판단합니다. 다른 자산과의 비교가 아니라, **절대적 기준(0)**과 비교합니다.

```
SPY 12개월 수익률 = +5% → 양수 ✅ → 투자 유지
SPY 12개월 수익률 = -3% → 음수 ❌ → 채권으로 전환
```

**사용 전략:** Dual Momentum(2단계), CAM, GTAA(SMA 기준)

---

### 유형 C: 듀얼 모멘텀 (Dual Momentum)

> "상대 모멘텀으로 종목을 고르고, 절대 모멘텀으로 안전장치를 건다"

**2단계 비교** — 상대와 절대를 결합한 방식입니다.

```
[1단계] 상대 모멘텀: SPY vs EFA → SPY 승리 (12M 수익률 더 높음)
[2단계] 절대 모멘텀: SPY의 12M 수익률 > 0인가?
  - YES → SPY 100% 보유
  - NO  → AGG(채권) 100% 전환
```

**사용 전략:** Dual Momentum, ADM, EDM, Papa DM, PDM

## 3. 방어 전환 트리거: 무엇이 위험 신호인가?

동적 전략의 핵심은 **"언제 공격에서 방어로 전환하는가?"**입니다. 전략마다 위험을 감지하는 방식이 다릅니다.

### 트리거 1: 절대 모멘텀 필터

| 전략 | 트리거 조건 | 행동 |
|------|------------|------|
| Dual Momentum | 승자의 12M 수익률 < 0 | 100% 채권 전환 |
| ADM | 승자의 평균 모멘텀 < 0 | 100% 채권 전환 |
| CAM | 다중 룩백 평균 < 0인 자산 | 해당 자산만 채권으로 교체 |

---

### 트리거 2: 전원 양수 규칙 (All-Positive Rule)

VAA의 핵심 규칙입니다.

```
공격 자산 4개의 13612W 스코어:
  SPY: +0.75  ✅
  EFA: +0.30  ✅
  EEM: -0.05  ❌ ← 하나라도 음수!
  AGG: +0.10  ✅

→ 하나라도 음수이면 → 100% 방어 자산 top-1로 전환
→ 전부 양수여야만 → 공격 자산 top-1에 투자
```

**엄격한 규칙:** 4개 중 1개만 음수여도 전체 포트폴리오가 방어로 전환됩니다. 이것이 VAA가 "Vigilant(경계하는)"인 이유입니다.

---

### 트리거 3: 카나리아 자산 (Canary Assets)

DAA, BAA가 사용하는 **조기 경보 시스템**입니다.

```
카나리아 자산: EEM(신흥국), AGG(채권)
이 2개의 13612W 스코어로 시장 건강 상태를 판단

[상태 1] 둘 다 양수 → 시장 건강 → 공격 자산 top-6 동일 비중
[상태 2] 하나만 음수 → 경고   → 50% 공격 + 50% 방어
[상태 3] 둘 다 음수 → 위험   → 100% 방어 자산 top-1
```

**카나리아를 쓰는 이유:** 신흥국 주식과 채권은 시장 스트레스에 가장 먼저 반응하는 자산입니다. 탄광의 카나리아처럼 위험을 조기에 감지합니다.

**DAA vs BAA 차이:**

| | DAA | BAA |
|--|-----|-----|
| 공격 시 선택 | top-6 동일비중 | top-1 집중 |
| 위험 성향 | 분산 (중위험) | 집중 (고위험) |
| 같은 카나리아 | EEM, AGG | EEM, AGG |

---

### 트리거 4: SMA 위치 필터

GTAA가 사용하는 방식입니다.

```
5개 자산 각각에 대해:
  현재가 > 10개월 SMA인가?

  미국 주식: 350 > SMA(340) → YES ✅ → 포함
  선진국:    120 > SMA(125) → NO  ❌ → 제외(현금)
  채권:      100 > SMA(98)  → YES ✅ → 포함
  리츠:      85  > SMA(90)  → NO  ❌ → 제외(현금)
  원자재:    45  > SMA(42)  → YES ✅ → 포함

→ 통과한 3개 자산에 동일비중(33.3%씩) 배분
→ 5개 모두 미달이면 100% 현금
```

---

### 트리거 5: 브레드스(Breadth) 기반

PAA, PDM, GPM이 사용하는 **시장 폭** 지표입니다.

```
12개 위험자산 중 양수 모멘텀 자산 수 = N

보호 계수(Protection Factor) 적용:
  채권 비중 = (12 - N) / 12 × 보호계수(2)

예시: N = 8 (12개 중 8개가 양수)
  채권 비중 = (12-8)/12 × 2 = 0.667 → 66.7% (상한 100%)
  → 실제로는 min(66.7%, 100%) = 66.7% 채권
  → 나머지 33.3%를 양수 모멘텀 자산 중 top-6에 배분

예시: N = 12 (전부 양수)
  채권 비중 = 0 / 12 × 2 = 0%
  → 100% 공격 자산 top-6 동일비중

예시: N = 2 (대부분 음수)
  채권 비중 = 10/12 × 2 = 1.667 → 100% (상한 적용)
  → 100% 채권
```

**특징:** 0% ↔ 100% 이분법이 아니라, **점진적으로** 방어 비중을 늘립니다.

---

### 트리거 6: 매크로 데이터 (거시경제 지표)

LAA가 유일하게 사용하는 방식입니다.

```
기본 배분: SPY 25% + GLD 25% + IWD 25% + IEF 25%

미국 실업률 체크:
  실업률이 상승 추세인가?
  - NO  → SPY 25% 유지
  - YES → SPY → SHY(단기채)로 교체

나머지 GLD, IWD, IEF는 항상 고정
```

**유일한 비가격 트리거:** 다른 전략은 모두 가격 데이터만 사용하지만, LAA는 거시경제 데이터(실업률)를 추가로 참조합니다.

## 4. 자산 선택 방식 비교

전략마다 "몇 개를 고르느냐"가 다르고, 이것이 위험도를 결정합니다.

| 선택 방식 | 전략 | 위험도 | 설명 |
|-----------|------|--------|------|
| **Top-1 집중** | Dual Momentum, VAA, BAA, ADM | 높음 | 1등 자산 하나에 100% 올인 |
| **Top-6 분산** | DAA, PAA | 중간 | 상위 6개에 동일비중 배분 |
| **Top-5 최적화** | AAA | 중간 | 상위 5개를 최소분산 비중으로 배분 |
| **SMA 통과 전체** | GTAA | 중간 | 조건 통과한 모든 자산에 동일비중 |
| **멀티레그 독립** | Papa DM, EDM | 높음 | 각 레그가 독립적으로 공격/방어 판단 |
| **점진 조절** | PAA, PDM, GPM | 중간~낮음 | 브레드스에 따라 방어 비중 점진 조절 |

## 6. 실제 앱에서의 비교 흐름

Allocate 앱에서 동적 전략이 실행되는 단계별 흐름입니다.

```
[사용자 입력]
  투자금: 10,000,000원
  전략: VAA
  유니버스: 한국 상장 ETF

        ↓

[1단계: 모멘텀 계산] — momentumCalculator.ts
  각 자산의 가격 히스토리에서 13612W 스코어 계산
  SPY(360750) → 0.75
  EFA(195930) → 0.30
  EEM(195980) → -0.05
  AGG(308620) → 0.10

        ↓

[2단계: 전략 규칙 적용] — strategyExecutor.ts
  VAA 규칙: 공격 4개 전부 양수인가?
  → EEM이 음수 → 방어 전환!
  → 방어 자산 중 top-1 선택
  SHY(214330): 0.08, IEF(308620): 0.10, LQD: 0.05
  → IEF(308620) 선택, 비중 100%

        ↓

[3단계: 매수 수량 계산] — allocationEngine.ts
  IEF 한국 ETF = KODEX 미국채10년선물 (308620)
  가격: 10,500원
  목표 금액: 10,000,000원
  매수 수량: floor(10,000,000 / 10,500) = 952주
  실제 투자: 952 × 10,500 = 9,996,000원
  잔여 현금: 4,000원

        ↓

[결과 화면 출력]
  ┌──────────────────────────────────────┐
  │ KODEX 미국채10년선물 (308620)         │
  │ 비중: 100.0%  │ 952주  │ 9,996,000원  │
  │ 잔여 현금: 4,000원                    │
  └──────────────────────────────────────┘
```

## 7. 핵심 개념 용어 정리

| 용어 | 영문 | 의미 |
|------|------|------|
| 모멘텀 | Momentum | 자산 가격의 추세 강도 |
| 룩백 | Lookback | 과거 몇 개월을 볼 것인가 |
| 리밸런싱 | Rebalancing | 포트폴리오를 전략 규칙에 맞게 재조정 |
| 카나리아 | Canary | 위험을 조기 감지하는 지표 자산 |
| 브레드스 | Breadth | 양수 모멘텀 자산의 수 (시장 폭) |
| 듀얼 모멘텀 | Dual Momentum | 상대 + 절대 모멘텀 결합 |
| SMA | Simple Moving Average | 단순 이동평균 |
| 13612W | Weighted Momentum | 1/3/6/12개월 가중 모멘텀 |
| Top-N | Top N Selection | 상위 N개 자산 선택 |
| 공격 자산 | Offensive Assets | 수익 추구 자산 (주식, 리츠 등) |
