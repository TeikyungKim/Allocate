# Google Play 앱 게시 체크리스트

> 작성일: 2026-03-29
> 현재 상태: 신원 확인 검토 중 (Google 승인 대기, 1~3일 소요)

---

## 1단계: 개발자 계정 인증 (Google Play Console)

- [x] 개발자 계정 생성 (등록비 $25 결제 완료)
- [x] 개발자 정보 입력 (이름, 주소, 이메일)
- [x] 신분증 업로드 (신원 확인 요청 완료)
- [ ] **신원 확인 승인 대기 중** — 승인 완료 시 이메일 수신
- [ ] 연락처 전화번호 인증 (신원 확인 완료 후 가능)

## 2단계: EAS Build (앱 빌드)

- [ ] Expo 계정 로그인 (`eas login --sso`)
- [ ] `eas.json` 빌드 프로필 확인
- [ ] 프로덕션 AAB 빌드 (`eas build --platform android --profile production`)
- [ ] 빌드된 `.aab` 파일 다운로드

## 3단계: Google Play Console — 앱 등록

- [ ] "앱 만들기" — 앱 이름(Allocate), 기본 언어(한국어), 앱 유형 선택
- [ ] 스토어 등록정보 작성
  - [ ] 앱 이름: Allocate (자산 배분)
  - [ ] 간단한 설명 (80자 이내)
  - [ ] 자세한 설명 (4000자 이내)
  - [ ] 앱 아이콘 (512x512 PNG)
  - [ ] 기능 그래픽 (1024x500 PNG)
  - [ ] 스크린샷 최소 2장 (폰 사이즈)
- [ ] 콘텐츠 등급 설문 작성
- [ ] 타겟 고객 및 콘텐츠 설정 (연령층)
- [ ] 개인정보처리방침 URL 등록
- [ ] 앱 카테고리 선택 (금융)
- [ ] 연락처 정보 (이메일, 웹사이트)

## 4단계: 앱 번들 업로드 및 심사 제출

- [ ] 프로덕션 트랙에 `.aab` 파일 업로드
- [ ] 출시 노트 작성 (한국어)
- [ ] 검토 제출
- [ ] Google 심사 통과 대기 (보통 수일 소요)

## 5단계: 출시 후

- [ ] Google Play에서 앱 정상 표시 확인
- [ ] AdMob 테스트 광고 → 실제 광고 ID로 전환
- [ ] EAS Secrets에 실제 AdMob ID 등록
