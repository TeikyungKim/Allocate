## 4. CSV 내보내기

**새 파일**: `src/services/csvExportService.ts`

### 기능
- 계산기 결과를 CSV 파일로 내보내기
- 웹: Blob download, 네이티브: expo-file-system + expo-sharing
- 한글 BOM 포함 (Excel 호환)
- 포함 정보: 티커, 이름, 자산군, 비중, 매수수량, 투자금액, 전략명, 유니버스, 잔여 현금

### 의존성
- `expo-file-system` 추가

### UI
- `CalculatorScreen`에 "CSV 내보내기" 버튼 추가 (포트폴리오 저장 옆)
