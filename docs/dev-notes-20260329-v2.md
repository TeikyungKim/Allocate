## 4. Firebase 동기화 개선 (완료)

### 변경 사항
- Google 로그인 성공 시 **자동 동기화** 실행
  - 기존: 로그인 후 수동으로 "지금 동기화" 버튼 클릭 필요
  - 개선: 로그인 직후 자동으로 `syncWithCloud()` 호출
- 동기화 실패 시 로그인은 유지 (에러 무시)

### Firebase 설정 상태
- `google-services.json` — 실제 프로젝트 (allocate-fd33e) 값으로 교체 완료
- `AuthContext.tsx` — webClientId 실제 값으로 업데이트 완료
- SHA-1 인증서 지문 Firebase에 등록 완료
- Google Authentication 활성화 완료

### 수정 파일

## 5. AdMob 상태 확인

### 현재 설정
| 항목 | 값 |
|------|-----|
| 앱 ID (Android) | ca-app-pub-9164387280735910~1904466035 |
| 배너 (전략) | ca-app-pub-9164387280735910/3565032060 |
| 배너 (포트폴리오) | ca-app-pub-9164387280735910/1346062833 |
| 전면 광고 | ca-app-pub-9164387280735910/3995337552 |

### 개발 빌드에서 광고가 안 나오는 이유
- `__DEV__` 모드에서는 Google 테스트 광고 ID 사용
- 테스트 광고도 Google Ads SDK 초기화 + 네트워크 필요
- **AdMob Console에서 앱이 활성 상태**여야 테스트 광고도 로드됨
- `adb logcat | grep -i admob`으로 로그 확인 권장

### 확인 필요 사항
1. [AdMob Console](https://admob.google.com) 에서 앱 등록 상태 확인
2. 앱 ID가 Console의 앱 ID와 일치하는지 확인
3. 광고 단위가 "활성" 상태인지 확인
